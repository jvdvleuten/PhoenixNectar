# Examples

Short reference snippets for common PhoenixNectar tasks.

- [BasicChat.swift](./BasicChat.swift): smallest connect, join, subscribe, and push flow
- [AuthenticatedSocket.swift](./AuthenticatedSocket.swift): auth token and connect params providers
- [AuthenticatedChat.swift](./AuthenticatedChat.swift): authenticated private-channel flow
- [ObserveConnectionState.swift](./ObserveConnectionState.swift): observe `connectionStateStream()`
- [ConnectionObserver.swift](./ConnectionObserver.swift): metrics hook plus connection state observation
- [StructuredLogger.swift](./StructuredLogger.swift): `OSLog.Logger` integration and sample output
- [TypedEventCatalog.swift](./TypedEventCatalog.swift): optional namespace helper for events and topics
- [BinaryUpload.swift](./BinaryUpload.swift): binary channel payloads

These files are snippets, not a standalone executable target.
